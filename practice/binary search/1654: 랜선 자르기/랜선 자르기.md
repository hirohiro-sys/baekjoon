## 問題リンク
https://www.acmicpc.net/problem/1654

## コード
```python
K, N = map(int, input().split())
A = [int(input()) for _ in range(K)]

# 1番長いラン線を得たいからmaxをendにする
s, e = 1, max(A)
while s <= e:
    mid = (s + e) // 2
    lan_cnt = 0
    for i in A:
        lan_cnt += i // mid
    # もしすでに目標本数に達していたら
    if lan_cnt >= N:
        # もっとラン線を長くできるからmidをsに
        s = mid + 1
    # 本数が足りないからもっとラン線を短く切るためにmidをeに
    else:
        e = mid - 1

print(e)
```
